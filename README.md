# AI Fashion Art Gallery

Welcome to the **AI Fashion Art Gallery** — a showcase of artificial intelligence-generated fashion artwork and creative designs.

## 🎨 Overview

This GitHub Pages site displays AI-generated fashion art, combining cutting-edge technology with creative expression. The gallery features responsive design and interactive features.

## 📁 Project Structure

```
AI-FASHION-ART-/
├── index.html      # Main gallery page
├── styles.css      # Styling and responsive design
├── script.js       # Interactive features and lightbox
└── README.md       # This file
```

## 🚀 Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Interactive Gallery**: Click on artworks to view in fullscreen lightbox
- **Modern UI**: Gradient backgrounds, smooth animations, and hover effects
- **Smooth Navigation**: Seamless scrolling between sections
- **Performance Optimized**: Fast loading and smooth interactions

## 📸 Adding Your Artwork

To add your AI-generated fashion art to the gallery:

1. **Create an `images/` folder** in the repository root
2. **Add your artwork images** (JPG, PNG, WebP formats recommended)
3. **Update `index.html`** in the gallery section:

```html
<div class="artwork-item">
    <img src="images/your-artwork.jpg" alt="Artwork Title">
    <p>Description of your AI fashion artwork.</p>
</div>
```

4. **Commit and push** your changes

## 🌐 View Your Gallery

Once GitHub Pages is enabled:
- **Project Site**: `https://echoesoffaith2-ux.github.io/AI-FASHION-ART-`

### Enable GitHub Pages:

1. Go to **Settings** → **Pages**
2. Select `main` branch as the publishing source
3. Select `/root` as the source directory
4. Click **Save**

Your site will be live within minutes!

## 🎯 Customization

### Change Colors and Branding

Edit `styles.css` to modify:
- Gradient colors in the `body` background
- Header styling
- Gallery item appearance
- Font families and sizes

### Update Content

Edit `index.html` to modify:
- Gallery title
- Navigation links
- About section text
- Contact information

### Add More Sections

Simply add new sections in `index.html` following this pattern:

```html
<section id="new-section">
    <h2>Section Title</h2>
    <p>Your content here...</p>
</section>
```

## 📱 Responsive Breakpoints

- **Desktop**: 1200px and above
- **Tablet**: 768px to 1199px
- **Mobile**: Below 768px

## 🔗 Interactive Features

- **Lightbox**: Click artwork to view fullscreen
- **Smooth Scrolling**: Click navigation links for smooth scrolling
- **Hover Effects**: Images zoom on hover with shadow effects
- **Touch Friendly**: All interactions work on touch devices

## 📝 Image Optimization Tips

For best performance:
- Use optimized images (compressed JPG or WebP)
- Recommended sizes: 400x500px to 1000x1200px
- Use descriptive file names
- Keep file sizes under 500KB each
